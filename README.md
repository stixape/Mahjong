# Mahjong

A Progressive Web App Mahjong solitaire game built with Vite, TypeScript, and PixiJS.

## Features

- Free Play and Challenge modes
- Progressive unlocks for layouts, tile styles, and backgrounds
- PWA install support for iPad home screen installation
- Offline-capable static deployment through GitHub Pages

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## GitHub Pages Deployment

This repository is configured to deploy automatically to GitHub Pages with GitHub Actions.

Published URL:

`https://stixape.github.io/Mahjong/`

After deployment completes, open that URL in Safari on iPad and use `Share -> Add to Home Screen` to install the PWA.
